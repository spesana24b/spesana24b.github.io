## API

Apakah kamu bertanya apa itu API pada situs?

Ya, API (Application programming interface) dimana sebuah enpoint yang menyediakan sebuah atau suatu fitur yang digunakan sebagai alat bertukar data atau informasi pada sebuah sistem web maupun aplikasi.
