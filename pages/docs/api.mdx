## API

Apakah kamu bertanya apa itu API pada situs?

Ya, API (Application programming interface) dimana sebuah enpoint yang menyediakan sebuah atau suatu fitur yang digunakan sebagai alat bertukar data atau informasi pada sebuah sistem web maupun aplikasi.

API pada situs sering kita temukan, namun sebagai developer biasanya mengumpetkan agar tidak dipakai dengan sembarangan.

Namun terkadang ada developer yang membuat api untuk memudahkan para developer untuk terhubung kepada aplikasi tersebut.

## API pada situs kelas

API ini dibuat untuk mempermudah digunakan untuk para pengembang, jika kamu menggunakan nodejs, kamu dapat menggunakan modulenya [disini !](/docs/nodejs-module)
