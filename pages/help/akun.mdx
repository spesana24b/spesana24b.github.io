## 📝 Sesi - Bantuan

## 📝 Bagaimana cara membuat halaman blog atau karya pada akun?

Masuk pada bar [Penyuntingan](https://spna.vercel.app/s?pg=akun&viewpage=1) lalu pilih type halaman, dan masukan judul untuk halaman.

Setelah itu kamu dapat menuliskan isi halaman tersebut, setelah serasa selesai menulis, kamu dapat posting halaman dengan informasi.

<Callout type="error" emoji="️🚫">
 Ingat!, jika kamu memaksa untuk membuat halaman tanpa informasi, mungkin saja akan bisa namun halaman tidak dapat dihapus dari sistem, dan harus menghapus lewat repository.
</Callout>

Setelah kamu klik buat halaman, tunggu sekitar 3 - 7 menit halaman akan muncul pada beranda atau informasi akun.

## 🤔 Bagaimana cara menghapus halaman?

Masuk ke [informasi atau pengaturan akun](https://spna.vercel.app/s?pg=akun), lalu klik tanda ![arrow_bottom_icon](/icon/arrow_bottom_icon.png), lalu pilih hapus halaman ini.

Tunggu Sekitar 3 - 7 menit halaman akan terhapus

## 📝 Sesi - Pertanyaan

## 🤔 Mengapa fitur ini ada?

Fitur ini sudah diajukan pada tanggal 5 September 2022 dan permintaan pada tanggal 3 Januari 2023, halaman ini digunakan untuk menunggah sebuah foto atau video (durasi 1 menit) ataupun posting instagram dan tiktok sebagai media untuk durasi video terpanjang.

Fitur ini dimudahkan untuk para siswa siswi alumni 24B untuk membuat halaman dan hanya perlu masuk dengan akun google

## ☠️ Apakah fitur ini aman?

Fitur ini belum tentu terverifikasi aman, [@ernestoyoofi](https://sp24b.vercel.app/s?na=13) sebagai admin server perlu mengembangkan keamanan dan mencoba untuk membuat repositori tersebut menjadi private.

## 📝 Apa saja yang ada difitur ini?

Tentunya adalah mengupload media seperti video atau gambar, namun juga bisa kamu dapat memposting postingan dari sosial media.

Fitur /posting digunakan untuk menunggah media atau assets saat ingin memberi video atau video pada halaman

Namun juga ada fitur untuk menyunting, yang berfungsi sebagai membuat, menggubah atau menghapus (hanya akun tersebut dapat menghapus) halaman yang diinginkan, kamu bisa pilih 2 type yaitu **karya** atau **blog**

## 🤔 Sebagai developer, apakah ada rest api untuk menggunakan sistem ini?

Ya, tentu ada donk!, fitur rest api digunakan untuk melakukan pertukaran data pada tampilan situs / website.

Kamu bisa mempelajari ini pada halaman [dokumentasi rest api](/docs/api).
